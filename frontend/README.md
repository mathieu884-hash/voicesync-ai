# VoiceSync AI Frontend

Modern React/Next.js frontend for the VoiceSync AI platform.

## Quick Start

```bash
npm install
npm run dev
```

## Structure

- `app/` - Next.js app directory (pages and layouts)
- `components/` - Reusable React components
- `lib/` - Utilities, API client, state management
- `styles/` - TailwindCSS configuration and globals
