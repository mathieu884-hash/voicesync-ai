## Getting Started

### Development

```bash
# Install dependencies
pip install -r requirements.txt
pip install -r requirements-dev.txt

# Configure environment
cp .env.example .env

# Initialize database
alembic upgrade head

# Run development server
uvicorn app.main:app --reload

# Run tests
pytest

# Run with Docker
docker-compose up
```
